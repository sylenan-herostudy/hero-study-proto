# Hero Study 2.0 — lo-fi HTML-прототип

Кликабельный wireframe нового маркетингового портала `hero.study` (все 13 разделов sitemap).
Назначение — проверить структуру, навигацию и логику сценариев до hi-fi-дизайна и сборки на Tilda.

## Как запустить

**Вариант 1 — двойной клик.** Откройте любой `.html` (например `index.html`) в браузере.
Работает без сервера: общая обвязка (шапка/футер/меню), калькулятор, модалки, формы — на чистом JS (Web Components с инлайн-шаблоном, без `fetch`).

**Вариант 2 — локальный сервер (рекомендуется).**
```bash
cd hero-study-proto
python3 -m http.server 8000
# открыть http://localhost:8000/
```

## Что это и чего здесь нет

Это **lo-fi wireframe**, а не финальный дизайн:
- палитра — серая шкала + один акцент;
- акценты 4 продуктов (green/violet/cyan/coral) показаны **текстовой меткой** `NOTE`, а не цветом;
- фото/видео/графики/3D/лого — штрихованные плейсхолдеры с подписью;
- реальны заголовки, навигация, подписи кнопок; тело абзацев — строки-плейсхолдеры.

Витрина всех компонентов и состояний — `styleguide.html`.

## Структура

- `index.html` — главная; `product-*.html` — 4 лендинга; `ss-*.html` — 2 подстраницы SS;
  `pricing / consultation / cases / about / partners / support` — разделы;
  `resources-* / help-*` — ресурсы; `account-*` — личный кабинет; `404 / thank-you / legal-*` — утилитные.
- `css/tokens.css` — дизайн-токены; `css/wireframe.css` — компоненты, состояния, адаптив.
- `js/nav-data.js` — единый источник меню; `js/components.js` — Web Components обвязки;
  `js/interactions.js` — весь интерактив; `js/calculator.js` — калькулятор; `js/i18n.js` — язык.

Меню правится в одном месте — `js/nav-data.js`.

## Чек-лист верификации

- Шапка/мега-меню (4 шт.), переключатель языка RU/EN/KZ/UZ, бургер на мобиле.
- Главная: видео-модалка, табы продуктов.
- Лендинг SS: интерактивная схема модулей (тултипы), табы «Преимущества».
- Цены: тоггл, калькулятор (слайдер/чекбоксы/живой пересчёт, ошибка при <100 студентов), FAQ.
- Консультация: мульти-степ форма с валидацией → `thank-you.html`.
- Партнёры: модалка-заявка. Поддержка: тикет-форма (состояния default/loading/success/error), чат-бот.
- Справочный центр: живой поиск + чипы. ЛК: дашборд → тикеты → документы (+ empty states).
- 404 / Спасибо / legal; футер и cookie-баннер на всех страницах.
- Брейкпоинты: **375 / 768 / 1280**.
